Gulp-pug-sass Boilerplate

  • Github
This is a boilerplate for web-frontend developer made by Minaki-web.

Commands

This is a basic command.

$npm start

You can run this command for building production.

$npm run build

You can run this command when you wanna only start web server and watching files.

$npm run watch

You can run this command when you wanna delete "dist".

$npm run reset

If you want more commands, you can see them in Github.

Tips

  • Store all images in under "src/img".
  • Store all favicons in under "src".
